TypeScript giúp bạn viết test code ít bug hơn nhờ kiểm tra kiểu dữ liệu ngay từ lúc code. Với QA học automation, đây là lợi thế lớn khi làm việc với API response, test data và Page Object.

Bash setup 
npm init -y
npm install -D typescript ts-node @types/node
npx tsc --init
TypeScript types.ts 
type TestStatus = "passed" | "failed" | "blocked";

const tcId: string = "TC-001";
const retryCount: number = 2;
const isCritical: boolean = true;

function printSummary(id: string, status: TestStatus) {
  console.log(`${id} => ${status}`);
}
💡
Union type rất hợp cho status, environment, browser name hay severity vì nó chặn ngay các giá trị sai.

Tuần 2 · Functions & Objects
📦
Object Types, Interfaces, Arrays, Function Signatures
interface · optional fields · readonly · map/filter
▼
Lý thuyết

Khi làm automation, bạn sẽ thường thao tác với object kiểu user, order, product, API response. TypeScript giúp định nghĩa trước cấu trúc của chúng bằng interface.

TypeScript models.ts 
interface LoginUser {
  username: string;
  password: string;
  role?: "admin" | "standard";
}

const users: LoginUser[] = [
  { username: "standard_user", password: "secret_sauce" },
  { username: "admin_user", password: "secret_sauce", role: "admin" },
];

function getAdminUsers(items: LoginUser[]) {
  return items.filter((user) => user.role === "admin");
}

Tuần 3 · Async & Modules
⏳
Async/Await, Promise Flow, Import/Export
fetch data · await · reusable utilities
▼
Lý thuyết

Playwright dùng bất đồng bộ rất nhiều. Nếu hiểu chắc async/await từ đầu, bạn sẽ đọc test flow và debug timing issue dễ hơn hẳn.

TypeScript api.ts 
export async function loadUsers() {
  const response = await fetch("https://reqres.in/api/users?page=1");
  const body = await response.json();
  return body.data;
}

export async function printFirstUser() {
  const users = await loadUsers();
  console.log(users[0]);
}

Lý thuyết

Phase này chuẩn bị tư duy cho Page Object Model. Nếu bạn quen với class, method và DOM query, việc lên Playwright sẽ mượt hơn nhiều.

TypeScript page-object.ts 
class LoginWidget {
  constructor(private rootSelector: string) {}

  getSubmitSelector() {
    return `${this.rootSelector} button[type="submit"]`;
  }
}

const loginWidget = new LoginWidget("#login-form");
console.log(loginWidget.getSubmitSelector());
