▼
Kiến thức thực chiến

Trong Angular app thật, DOM render ra thường không phản ánh “business structure” một cách sạch sẽ. IDs như mat-input-4, classes kiểu ng-star-inserted hay cdk-overlay-pane thường không bền. Vì vậy, đọc source component và tìm semantic container là kỹ năng rất đáng học.

TypeScript angular-locator.ts 
// ❌ Selector bám generated DOM
page.locator(".ng-star-inserted .mat-mdc-select-trigger");

// ✅ Bám vào contract ổn định
const panel = page.locator("[data-testid='user-search-panel']");
await panel.getByLabel("Department").click();
await page.getByRole("option", { name: "Finance" }).click();

Tuần 15 · Table, Overlay, Virtual Scroll
📊
Enterprise Tables, Filters, Sticky UI, Async Rendering
business key assertions · pagination · scroll container · lazy rows
▼
Kiến thức thực chiến

Ở enterprise UI, bạn hiếm khi test một row “đứng yên” trong DOM. Table có thể filter server-side, sort, sticky header, virtual list, infinite scroll hoặc loading nhiều bước. Assertion tốt phải dựa trên business key chứ không phải vị trí.

TypeScript table-search.spec.ts 
const responsePromise = page.waitForResponse("**/api/orders*");
await page.getByRole("button", { name: "Search" }).click();
await responsePromise;

const table = page.locator("[data-testid='orders-table']");
await expect(table).toContainText("ORD-10025");

Tuần 16 · Debugging & Dev Contract
🤝
Test Hooks, Source Review, Flaky Prevention
data-testid requests · loading state contract · sustainable automation
▼
Kiến thức thực chiến

Một QC automation mạnh không cố “hack selector cho chạy”, mà biết khi nào cần đọc source, khi nào nên yêu cầu dev thêm data-testid, khi nào cần loading state rõ ràng hơn, và khi nào một control nên được refactor để testable.

✅
Nếu control quá phức tạp, hãy mô tả rõ cho dev: element nào là trigger, list render ở đâu, text nào đại diện business value, và test hook nào sẽ giúp suite bền hơn.
⚠️
Một dấu hiệu tốt để push back: selector hiện tại bắt buộc phải bám vào generated id/class, overlay không có accessible name, hoặc loading state biến mất quá sớm so với API/data thật.
